## ðŸ†˜ Troubleshooting

### Game Won't Start
1. Make sure you have Java 17+ installed
2. Try running `update_modpack.bat` to get the latest files
3. Check if your graphics drivers are up to date

### Update Script Fails
1. Make sure you have internet connection
2. Try running as Administrator
3. Temporarily disable antivirus if it blocks the script

### Performance Issues
1. Allocate more RAM in Prism Launcher settings (6GB+ recommended)
2. Make sure you're running on an SSD if possible
3. Close other applications while playing

### VR Not Working
1. Make sure your VR headset is connected before launching
2. Install SteamVR if you haven't already
3. Check that Vivecraft is properly installed

### Server Connection Issues
1. Verify you're using the correct server IP
2. Check if you're whitelisted on the server
3. Make sure you have the latest modpack version
4. Contact PlepperGuy if you can't connect
